⚙️ Upgrade Notes

To make this fully real

You will add Netlify functions for OCR, translation, and AI interpretation so your private API keys never appear in the browser.

Recommended backend functions:

netlify/functions/ocr.js
- Receives image
- Runs Google Cloud Vision DOCUMENT_TEXT_DETECTION
- Returns extracted text + confidence

netlify/functions/translate.js
- Receives extracted text
- Runs Google Cloud Translation to English
- Returns detected language + translation

netlify/functions/oracle.js
- Receives OCR text, notebook notes, image observations, or signal transcript
- Runs AI interpretation
- Returns symbolic reading with confidence warning

netlify/functions/notebook-oracle.js
- Receives any image
- Runs vision analysis or OCR if available
- Returns: visible elements, extracted words if any, symbolic interpretation, practical notes

- Browser-only canvas tool already works without API keys
- Future upgrade: send forged image to AI image model for stronger stylization
